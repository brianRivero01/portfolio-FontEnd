RÉPLICA DE SITIOS WEB

En mis tiempos libres me gusta elegir páginas de aplicaciones web e intentar hacerlas por mí mismo, incluyendo todo lo que es su contenido y las funcionalidades que estas páginas ofrecen. Esto me ha servido un montón y me ha ayudado a poner en práctica mis conocimientos así como también aprender funcionalidades nuevas para aplicar a la hora de estar haciendo un sitio web. Dejo acá un par de ejemplos:

PORTFOLIO WEB

Proyecto final de la segunda etapa del curso de Argentina Programa. Portfolio personal donde expongo toda información profesional realizada, ya sean cursos, experiencias profesionales, formación académica, así como también, gráficos acerca de distintos conocimientos en programación. Contiene botones de edición que se muestran al iniciar sesión y que me sirven para editar información o imágenes que se hayan incluido dentro del portfolio. Este inicio de sesión se valida a través de un usuario y contraseña cuyos datos, con anterioridad, se guardaron en una base de datos. En este proyecto se utilizaron tecnologías como Angular, Spring Boot y MySQL.
